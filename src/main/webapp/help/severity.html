Select the severity level from the drop-down.
The failure condition by a vulnerability severity fails a build if a vulnerability with a specified or higher severity is found.
For example: If the severity level selected is '1' then the build will fail if a vulnerability with severity 1 or greater than 1 that is (1,2,3,4,5) is found in the scan.
NOTE: Severity 1 rating is least severe and severity 5 is most severe.
